## このアプリでできること

こっそりタイムラインを見るのに向いた Mastodon リーダーです。

- 目立ちにくい: UI が軽く、フル機能の SNS サイトよりも小さなツールウィンドウのように見えます。
- よりプライベート: 通常の Web ページのように、アバターや表示名、サイドバー、各種ソーシャル要素を大きく表示しません。
- キーボード優先: Vim 風の操作が好きな人に向いていて、頻繁にマウスへ手を伸ばさずに使えます。
- ワンキー切り替え: いちばん重要な点として、グローバルショートカットでウィンドウを即座に表示 / 非表示できます。

## Vim ユーザー向けの操作

標準では次の操作に対応しています。

- `j` / `k`：下 / 上にスクロール
- `u` / `d`：半ページ上 / 下へ移動
- `gg`：先頭へ移動
- `G`：末尾へ移動
- `r`：タイムラインを更新

## ワンキー切り替え

このアプリはグローバルショートカットに対応しています。

既定のショートカット:

```text
CommandOrControl + D
```

## インスタンス URL と Token の設定方法

初回利用時は設定画面で次の 2 つを入力してください。

- `Instance URL`
- `Access Token`

### インスタンス URL の入力方法

自分の Mastodon インスタンスの URL を入力します。たとえば:

```text
https://mastodon.social
```

`https://` を付けずに入力しても、アプリが自動で補完します。

### Token とは

`Access Token` は、このアプリがあなたのタイムラインを読むための鍵のようなものです。

現在は主に `Home` タイムラインを読むために使われ、ここで改めて Web ページ経由のログインをする必要はありません。

### Token の取得手順

1. 自分の Mastodon インスタンスを開き、アカウントにログインします。
2. 次の画面へ進みます:

```text
Preferences -> Development -> New Application
```

3. 新しいアプリを作成します。
4. 名前は `TouchFish` などにします。
5. `Scopes` で少なくとも `read` を有効にします。
6. 作成後、そのアプリの詳細ページを開きます。
7. `Access Token` をコピーします。
8. TouchFish に戻り、設定画面の `Access Token` に貼り付けます。

入力が終われば、`Home` タイムラインへ切り替えられます。
